1) Write a query that lists each order number followed by the name of the customer
who made the order.
	
	select orders.onum,customers.cname from orders,customers where orders.cnum = 	customers.cnum; 

2) Write a query that gives the names of both the salesperson and the customer for
each order along with the order number1.

	 select orders.onum ,customers.cname,salespeople.sname from 	orders ,customers,salespeople where orders.snum = customers.snum and customers.snum = 	salespeople.snum;

3) Write a query that produces all customers serviced by salespeople with a
commission above 12%. Output the customer’s name, the salesperson’s name,
and the salesperson’s rate of commission.

	select customers.cname ,salespeople.sname,salespeople.comm from customers,salespeople 	where salespeople.comm > 0.12;
