1) Write a query that selects all customers whose ratings are equal to or greater than
ANY of Serres’.

	select * from customers where snum >= any (select snum from salespeople where city 	="serres");

2) Write a query using ANY or ALL that will find all salespeople who have no
customers located in their city.
	
	select * from salespeople where city = any (select city from customers);

3) Write a query that selects all orders for amounts greater than any for the
customers in London.

	select * from orders where amt => any (select o.amt from orders o,customers c where o.snum = c.snum and c.city = "london");

4) Write the above query using MIN or MAX. 

select * from orders where amt > (select min(o.amt) from orders o,customers c where o.snum = c.snum and c.city = "london");

select * from orders where amt > (select max(o.amt) from orders o,customers c where o.snum = c.snum and c.city = "london");
